About Me

I'm a Technical Artist with a decade of industry experience across gaming and XR technologies. Throughout my career at Meta, Apple, and Sledgehammer Games, I've focused on creating efficient workflows and solving complex technical challenges to empower creative teams and deliver exceptional user experiences.

In a typical work day, I develop robust pipelines, optimize performance, and foster an environment of learning and collaboration. I excel at translating technical requirements between disciplines, helping engineers understand artistic needs and helping artists work effectively within technical constraints.

I'm passionate about collaborative problem-solving and work closely with team leads to identify and address technical challenges before they impact production milestones. My experience and expertise allows me to provide valuable technical leadership which fuels my primary goal of helping others succeed.

I approach each challenge with a focus on both technical excellence and effective teamwork.

Technical Projects

Performance Automation Framework

📊

Developed a comprehensive instrumentation and automation framework for VR applications that captured performance metrics across the entire codebase. The system automatically tracked framerate, memory usage, load times, and other critical metrics to identify bottlenecks.

Python Performance VR

Performance Visualization Dashboards

📈

Created interactive performance dashboards that visualized metrics from our VR applications, helping teams tackle regressions more effectively. These dashboards tracked trends over time and enabled drill-down analysis by world, player count, and hardware configuration.

Data Visualization Python Performance

Procedural Scene Generation

🏙️

Developed procedural scene generation systems for Apple Vision Pro training data. Created parametrically controlled environments with variable lighting, object placement, and appearances that generated thousands of synthetic training scenarios.

Python Maya Procedural

Asset Validation QA Tools

🔍

Built comprehensive QA tooling for Apple that accelerated asset integration by validating models, textures, and metadata. Features included automatic thumbnail generation, LOD creation, metadata validation, and seamless outsource integration.

Python Pipeline Quality Assurance

LOD Generation Pipeline

💎

Integrated a proprietary LOD generation algorithm into Sledgehammer Games' asset pipeline for Call of Duty WWII, replacing manually authored LODs. This system automated the creation of optimized mesh levels, significantly reducing artist workload while maintaining visual fidelity.

C++ C# Optimization

Edge Cleanup Tool

✂️

Created an optimization tool that identifies and removes unnecessary edges in 3D models. The script intelligently detects edges that don't add visual detail to the model but consume processing power, providing significant polygon reduction with no visual impact.

Python Maya Optimization
